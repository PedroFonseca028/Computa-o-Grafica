# Classificação de Imagens com CNN - Animals-10

Projeto de classificação de imagens de animais utilizando TensorFlow + Keras.

## Objetivo
Comparar duas arquiteturas de CNN para classificar imagens do dataset Animals-10.

## Tecnologias
- Python
- TensorFlow
- Keras
- Matplotlib
- Scikit-learn
- VSCode

## Estrutura
- data/raw-img/: dataset original
- data/split/: treino, validação e teste
- models/: modelos salvos
- results/: gráficos e métricas
- src/: códigos principais
- notebooks/: análises e testes
- slides/: apresentação

## Como executar
1. Instale as dependências:
   pip install -r requirements.txt

2. Execute o projeto:
   python main.py
